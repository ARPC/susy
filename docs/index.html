    susy

    $susy: () !default;
    View source

    Description

    Custom user configuration map for Susy

    Type

    Map

    Map structure

    Map keyNameMap keyDescriptionMap keyTypeMap keyValue
    columns

    Description of the columns in a grid, their number, realtive sizes (if asymmetrical), and optional static widths

    		Number or List4
    gutters

    Ratio of a gutter to a column, or static width of a single gutter

    		Number0.25
    spread

    Spread of an element across adjacent gutters: either narrow (none), wide (one), or wider (two)

    		String'narrow'
    container-spread

    Spread of a container around adjacent gutters: either narrow (none), wide (one), or wider (two)

    		String'narrow'

    Example

    All available options (to their default value)

    $susy: (
  'columns': 4,
  'gutters': 0.25,
  'spread': 'narrow',
  'container-spread': 'narrow',
);

    susy-span (aliased as span )

    @function susy-span($span, $config: ()) { ... }
    View source

    Description

    This is the primary function in Susy — used to return the width of any grid-span — such as the width property of grid elements, or margin and padding to push, pull, and pad your elements (un-prefixed alias span() is available unless you import susy/prefix explicitly)

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $span

    Shorthand expression to define the width of the span — containing a unitless column-span; 'of $n' for available grid columns [optional]; 'at $n', 'first', or 'last' for location on asymmetrical grids; 'narrow', 'wide', or 'wider' for optionally spreading either the span (before of) or columns (after of) over adjacent gutters; and 'set-gutters $n' to override global gutter settings.

    		List—none
    $config

    Optional map of Susy grid configuration settings

    		Map()

    Returns

    Length —

    Fluid % length when no column-width is given, or a static length in the same units as column-width

    Example

    Span half the grid

    .foo
  width: susy-span(6 of 12);
}

    Span a specific segment of asymmetrical grid

    .foo {
  width: susy-span(3 at 3 of (1 2 3 5 8));
}

    Requires

    • [function] susy-settings
    • [function] su-span

    Used by

    • [function] span
